# Project 3 Summary

Project 3 uses a genetic algorithm (GA) to optimize a 15-parameter swarm control law for 3D agents tasked with mapping stationary targets while avoiding obstacles, inter-agent collisions, and leaving the domain.

Deliverables are generated via `test_scripts.ipynb` and saved into `projects/project3/figures/` (convergence plot, star-metric plots, MP4 animation, and 5 snapshot frames).
